# Proyecto final

## Instalar

```bash
npm create vue@latest
```

> typescript, pinia, Router

## Vuetify + iconos

### Instalar

```bash
npm i vuetify @mdi/font
```

### Configurar vuetify

1. Crear archivo `vuetify.ts` en la carpeta `src/plugins`
2. Configurar el archivo `main.ts`

## Instalar Axios
